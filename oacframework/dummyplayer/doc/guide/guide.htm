Dummy Player

A jQuery plugin for simulating a player

Dummy Player looks like a real player and has got the same behaviour. It does not show any video, only displays the current time position and some controls.

Features

  • Customizable player size.
  • Play, Pause, Rewind and Forward controls.
  • User customizable function on playhead updating.
  • Playhead is easy to get and set.
  • Coordinates are easy to get.

Usage

  1. Download the latest version of Dummy Player below and jQuery 1.3.2 or later
  2. Setup the script references in the header: 
    <head>
        ...
        <script type="text/javascript" src="js/jquery-1.3.2.js"></script>
        <script type="text/javascript" src="js/dummy-player.1.0.js"></script>            
        ...
    </head>
                           
  3. Add a div container for your player. It must have a "dummyplayer" css class. Size is customizable by setting proper css values on this container: 
    <div class="dummyplayer"> style="width:300px; height:200px;"></div>
