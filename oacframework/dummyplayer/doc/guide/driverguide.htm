OAC Dummy Player Driver

A driver to control Dummyplayer from OAC framework.

This driver access Dummy Player and allows third parts control it through a common OAC interface. It must use with OAC framework.

Features

  • Access to play, pause, rewind and forward functions
  • Retrieve coordinates of video
  • Retrieve size of video
  • Set a user callback on each playhead update
  • Get a list of available players on a page

Usage

  1. Download the latest version of OAC Dummy Player Driver below.
  2. Setup the script references in the header: 
    <head>
        ...
        <script type="text/javascript" src="js/jquery-1.3.2.js"></script>
        <script type="text/javascript" src="js/dummy-player.1.0.js"></script>    
        <script type="text/javascript" src="js/oac-framework-1.0.js"></script>
        <script type="text/javascript" src="js/oac-dummyplayer-drv-1.0.js"></script>
        ...
    </head>
                           
  3. Add your dummy player: 
    <div class="dummyplayer"> style="width:300px; height:200px;"></div>
  4. Call video functions:
  5. Get video details: 
    OAC_Controller.player().getPlayhead()
